f3-redirect
=========

Adds a 404 tracking, and 301 redirects for 404s

### Getting Started

```
Add this to your project's composer.json file:

{
    "require": {
        "dioscouri/f3-redirect": "v0.9.0"
    }
}
```

### Use case

Keeping track of 404 URLS can be a nightmare, this makes it less of a nightmare
